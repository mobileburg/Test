# Презентация: Внедрение Битрикс24 для B2B «под ключ»

Деловая презентация в стиле **Bitrix24 / Русский Бит (crmnow.ru)**.

## Файлы

| Файл | Описание |
|------|----------|
| `index.html` / `slides.html` | Экранная версия (8 слайдов 16:9) |
| `print.html` | Версия для печати / PDF |
| `Bitrix24_B2B_Russkiy_Bit.pdf` | Готовый PDF |
| `assets/` | Сгенерированные иллюстрации |
| `screenshots/` | Превью слайдов |

## Просмотр

```bash
cd presentation
python3 -m http.server 8765
# открыть http://127.0.0.1:8765/
```

## Стиль

- Цвета Bitrix24: `#2FC7F7`, `#409EEF`, `#0066A1`, акцент `#BDF300`
- Шрифты: Montserrat + Open Sans
- Инфографика, кейсы, этапы, CTA на crmnow.ru
